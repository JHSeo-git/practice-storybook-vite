# practice-storybook-vite

Practice for React + Vite + Storybook

## vite + storybook init

### 1. create vite app

```bash
yarn create vite

# [project-name]
# - React
# - TypeScript + SWC
```

### 2. install dependencies

```bash
cd [project-name]
yarn install
```

### 3. init storybook

```bash
npx sb init --builder @storybook/builder-vite
```
