# Practice Storybook

> with tailwindcss, Vite, TypeScript

스토리북과 tailwindcss, Vite, TypeScript를 함께 적용한 데모 프로젝트입니다.

스토리북은 비지니스 로직, 데이터, 컨텍스트에서 독립된 상태에서 쉽게 UI 컴포넌트를 만들 수 있습니다.
개발하는 동안 **stories**에 이러한 UI 상태를 저장하여 개발, 테스트 또는 QA 중에 사용할 수 있습니다.

왼쪽 메뉴에서 예제 스토리를 탐색하고 작동방식을 확인해보세요.

> **Note**  
> atomic 컴포넌트로 시작하여 pages로 끝나는 일련의 프로세스인 <a href="https://componentdriven.org" target="_blank"><strong>component-driven</strong></a> 프로세스로 UI를 구축하는 것을 추천합니다.
