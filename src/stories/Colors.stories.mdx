# Colors

Colors는 애플리케이션 디자인에 사용할 수 있는 색상을 제공합니다.

`vars`를 사용하여 색상을 사용할 수 있습니다.

```js
import vars from '@design-system/vars';

// use the semantic colors
const primaryColor = vars.$color.semantic.primary9;

// use the scale colors
const blueColor = vars.$color.scale.blue11;
```

Colors는 크게 2개로 나뉘어집니다.

- Scale Colors: 강도를 나타내는 색상입니다. (red1, red2, red3, red4, ...)
- Semantic Colors: 의미를 가지고 있는 색상입니다. (primary, success, ...)

## Scale

12단계 Scale Colors를 제공합니다. 총 6개의 색상을 제공합니다.
