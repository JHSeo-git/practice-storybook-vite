# Installation

How to install this system.

<Callout type="warning" emoji="⚠️">
  This is not production ready. It is for practice only.
</Callout>

## Git clone

Run the following command to create a new project.

```bash
git clone https://github.com/JHSeo-git/practice-storybook-vite
```

## Existing project

If you already have a project, you can install Storybook in the following way.

First, install Tailwind CSS.

{/* prettier-ignore */}
<Tabs items={['pnpm', 'npm', 'yarn']}>
  <Tab>
    ```bash
    pnpm add -D tailwindcss postcss autoprefixer
    ```
  </Tab>
  <Tab>
    ```bash
    npm install -D tailwindcss postcss autoprefixer
    ```
  </Tab>
  <Tab>
    ```bash
    yarn add -D tailwindcss postcss autoprefixer
    ```
  </Tab>
</Tabs>
